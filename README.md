# Product Management System (CLI)

[![Python Version](https://img.shields.io/badge/python-3.x-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Um sistema prático e interativo de controle de estoque e gerenciamento de produtos desenvolvido em Python. O aplicativo roda diretamente no terminal (CLI) e foi projetado com foco em usabilidade, organização de dados em memória e robustez no tratamento de entradas do usuário.

---

## 🚀 Funcionalidades

O sistema oferece uma interface de menu completa que simula as operações essenciais de um gerenciamento de estoque (**CRUD**):

1. **Adicionar Produto:** Permite cadastrar um novo produto informando nome, quantidade em estoque e preço unitário.
2. **Listar Produtos:** Exibe de forma organizada todos os itens cadastrados no sistema com formatação de moeda (R$).
3. **Buscar por Nome:** Localiza um produto específico e exibe instantaneamente seus detalhes de estoque e preço.
4. **Editar Informações:** Permite atualizar a quantidade ou o preço de um produto já cadastrado de maneira flexível.
5. **Tratamento de Exceções:** Proteção ativa contra erros de digitação (ex: digitar texto onde o sistema espera um número), garantindo que o programa não feche inesperadamente.

---

## 🛠️ Tecnologias Utilizadas

* **Python 3.x:** Linguagem principal do projeto.
* **Estruturas de Dados:** Listas compostas (matrizes em memória) para armazenamento dinâmico dos dados.
* **Controle de Fluxo e Exceções:** Blocos `try/except` e estruturas condicionais (`if/else`) para validação rigorosa de dados.
